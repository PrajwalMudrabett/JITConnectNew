## рџЋЇ How to Download Images from JIT Website

### Method 1: Right-Click Save
1. Go to https://jyothyit.ac.in/
2. Right-click on any image
3. Select "Save image as..."
4. Save to the appropriate folder

### Method 2: Browser DevTools
1. Open https://jyothyit.ac.in/
2. Press F12 (Developer Tools)
3. Go to "Network" tab
4. Refresh page
5. Filter by "Images"
6. Right-click on image URLs and "Open in new tab"
7. Save the images

1. Take high-quality screenshots of the website
2. Crop to desired sections
3. Save in appropriate folders
