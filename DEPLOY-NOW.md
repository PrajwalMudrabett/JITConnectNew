## 🎯 Deploy to Vercel (3 Steps - 5 Minutes)

### Step 1: Go to Vercel
Open: https://vercel.com/new

### Step 2: Import Repository
1. Click **"Import Git Repository"**
2. Find and select: **"JITConnectNew"**
3. Click **"Import"**

### Step 3: Configure & Deploy
**Build Settings:**
- Framework Preset: **Vite**
- Root Directory: **jitconnect-react**
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`

**Environment Variables:**
Click "Environment Variables" and add:
```
Name: VITE_API_URL
Value: https://jitconnectnew.onrender.com/api
```

Click **"Deploy"** button!

---

## 🔧 After Deployment (1 Minute)

### Update Backend CORS
1. Copy your Vercel URL (e.g., `https://jitconnect-abc123.vercel.app`)
2. Go to: https://dashboard.render.com
3. Select your backend service
4. Go to **Environment** tab
5. Update `CLIENT_URL` with your Vercel URL
6. Click **"Save Changes"**

Backend will automatically restart.

## 🐛 Troubleshooting

**Build Failed?**
- Check Root Directory is set to `jitconnect-react`
- Verify environment variable is added

**CORS Error?**
- Update CLIENT_URL on Render
- Restart backend service

**API Timeout?**
- Wait 30 seconds (Render waking up)
- Refresh the page
